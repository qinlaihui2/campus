## 项目经验

### 校圈 CampusHub — AI 校园综合平台（独立开发）

**时间**：2025.10 - 至今
**角色**：全栈独立开发
**线上地址**：http://120.55.76.100

**项目简介**：
基于 RAG（检索增强生成）技术的 AI 校园综合平台，集成 DeepSeek 大模型实现智能问答。覆盖 AI 对话、课程学习、问答广场、知识库管理、私信 IM、二手交易市场、失物招领、公告系统等 16 个业务模块。Spring Boot 3.4 + Vue 3 全栈架构，Docker Compose 部署。

**技术栈**：
Spring Boot 3.4 / Spring Security 6 / JWT / MyBatis Plus 3.5 / Vue 3 + TypeScript / Vite 6 / Pinia / Element Plus / LangChain4j 1.0 / DeepSeek API / pgvector / PostgreSQL 16 / MySQL 8.0 / Redis + Redisson / RabbitMQ / MinIO / WebSocket / Docker / Nginx / Git

**核心亮点**：

1. **RAG 智能问答引擎**：从文档上传 → 文本解析 → Chunk 分割（500/50）→ all-MiniLM-L6-v2 本地向量化 → pgvector 向量检索（cosine_similarity）→ Prompt 组装 → DeepSeek SSE 流式输出。全链路自研，不调第三方 RAG 服务。

2. **14 模块 Maven 多模块架构**：按业务领域拆分，单向依赖无循环引用。campus-common 零业务依赖，模块间通过接口解耦，可快速拆分为微服务。

3. **双 Token 认证 + 限流**：Access Token（30min）+ Refresh Token（7d），JWT 拦截器 + ThreadLocal 上下文。Redisson RRateLimiter + AOP 切面实现声明式接口限流。

4. **WebSocket 实时私信**：会话式私信 IM，支持文本/表情，3 秒自动重连，在线状态推送。

5. **Toggle 模式点赞/收藏**：UNIQUE INDEX 防重复 + 冗余计数字段避免 JOIN + 数据库原子操作防并发竞态 + 前端即时响应无需刷新。

6. **RabbitMQ 异步文档处理**：大文件上传 → MinIO 存储 → MQ 解耦 → Consumer 异步解析 → 向量化入库，HTTP 请求不阻塞。

7. **32 个自动化测试**：JUnit 5 + Mockito + AssertJ + MockMvc，覆盖 Service 层和 Controller 层集成测试。

**工程能力**：
- CI/CD：Git push → 服务器 git pull → Maven 增量编译 → npm build → Nginx 热更，一行命令部署
- 缓存反模式实战：公告模块因 Redis 缓存导致删除后数据不一致，移除缓存回归简单 CRUD，总结了缓存适用场景
- 全栈开发：一个功能从数据库建表 → Entity → Mapper → Service → Controller → 前端 API → UI 组件 6 层贯通
