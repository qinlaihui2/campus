校园办事流程指南

校园卡办理与挂失流程
1. 新生入学时统一采集照片并发放校园卡，无需单独办理。
2. 校园卡丢失后，请立即通过校园APP或自助服务终端进行挂失，防止被盗刷。
3. 挂失后携带身份证到校园卡服务中心（行政楼一楼）补办，工本费20元。
4. 补办新卡后，原卡余额自动转入新卡，门禁权限需到宿管处重新授权。
5. 校园卡服务中心工作时间：周一至周五 9:00-12:00，14:00-17:00。

学生证补办流程
1. 下载并填写《学生证补办申请表》，由辅导员签字确认。
2. 携带申请表、身份证复印件、一寸照片2张，到教务处学籍科办理。
3. 缴纳工本费10元，领取回执单。
4. 7个工作日后凭回执单领取新学生证。
5. 学生在校期间最多补办2次学生证。

成绩单及在读证明办理流程
1. 在校生可通过教务系统自助打印成绩单和在读证明，图书馆一楼设有自助打印机。
2. 如需加盖公章的成绩单，请到教务处办理，每份收费2元。
3. 毕业生成绩单到档案馆（图书馆五楼）办理，需提前1个工作日预约。
4. 英文成绩单需额外申请翻译服务，每份收费5元，5个工作日内完成。

校园网接入流程
1. 连接校园Wi-Fi（SSID：Campus-WiFi）。
2. 打开浏览器访问任意网页，自动跳转到认证页面。
3. 使用学号和统一身份认证密码登录。
4. 每人每月免费流量50GB，超出后限速至1Mbps。
5. 如遇网络故障，可拨打网络中心服务热线：020-12345678。

医疗报销流程
1. 在校医院就诊：直接使用校园卡结算，个人承担20%，学校补贴80%。
2. 校外就医：需先经校医院开具转诊单，方可到定点医院就诊。
3. 报销材料：转诊单、病历、医疗发票、费用清单。
4. 每月1-5日将报销材料交至校医院医保办公室。
5. 审核通过后，报销款于次月15日打入学生银行卡。
